URLs

Over the coming months we will be switching to a new URL format!
For the time being, all old URLs will get a 301 redirect to the new format.
  1. s.mlcdn.co
  2. library
  1. s.mlcdn.co
  2. library
  3. v=version

Parameters

Name Description
library The library you wish to use.
version The version you wish to receive, can be a range. If not provided, you will receive the latest version. Note: this will not always be as fast as providing no version, as the request may not yet be cached.

Examples

  1. s.mlcdn.co
  2. jquery.js
  1. s.mlcdn.co
  2. jquery.js
  3. v=3.x.x

Headers

Reaponse Headers

Name Description
x-version The version of the library that you are receiving.

Errors

If you encounter these errors in production, you are doing it wrong.

Format


{
  "code": 1,
  "message": "library 'foo' not found"
}

Error Codes

Code Status Code Description
1 404 You passed a library that doesn't exist.
2 404 You passed a version which matches nothing we have. You can always add more versions via GitHub!
3 400 You passed an invalid version string.
